快捷键:

	高亮：ctrl+h
	加粗：ctrl+b
	倾斜：ctrl+i
	删除线：ctrl+d
	代办事项：ctrl+l
	代码： ``` code  ```
	表格：|aaa|aaa|aaa|
			|:---|---:|---|      ：代表对齐
			|asd|asd|asd|
	缩进：tab
	点排列：- 
	序号排列： 1. 
	链接：[标题](网址)
	分割线：***
	颜色：<font color=red>   </font>
	文本：`asd`
	行内公式： $公式$
	行间公式:  $$ 公式 $$
